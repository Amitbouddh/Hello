How to set up:

  1. Upload your PDF to GitHub repository
  2. Click on your PDF file, then click "Raw" button
  3. Copy the URL from address bar
  4. Replace YOUR_PDF_URL_HERE in the code with your URL
  5. Change download="Your-Document.pdf" to your PDF name

Example URL: https://raw.githubusercontent.com/username/repo/main/file.pdf

PDF will download directly without opening in browser
